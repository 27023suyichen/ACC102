## Stock Performance Analyzer - ACC102 Mini Assignment (Track 4)

**Duration:** 2-3 minutes

---

### Opening (0:00 - 0:20)

**Visual:** Show the Streamlit app homepage

**Narration:**
"Welcome to the Stock Performance Analyzer, an interactive data analysis tool developed for the ACC102 Mini Assignment Track 4. This tool helps individual investors and finance students analyze and compare stock performance across different companies and sectors."

---

### Problem Definition (0:20 - 0:40)

**Visual:** Show the sidebar with stock selection options

**Narration:**
"The tool addresses a common challenge: comparing stock performance efficiently. Users can select from seven major US stocks across different sectors - Technology, Finance, Consumer, and Healthcare. The analysis period can be adjusted from one month to two years."

---

### Key Features Demo (0:40 - 1:30)

**Visual:** Navigate through different tabs

**Narration:**
"Let me demonstrate the key features. First, the Price Trends tab shows normalized performance comparison, allowing users to see how different stocks have performed relative to each other. The moving averages and Bollinger bands provide technical analysis insights."

**Visual:** Show the Statistics tab

**Narration:**
"The Statistics tab provides comprehensive descriptive statistics including start and end prices, mean, standard deviation, and total returns. Users can download this data as CSV files for further analysis."

**Visual:** Show the Risk Analysis tab

**Narration:**
"The Risk Analysis tab displays volatility metrics, Sharpe ratios, and a risk-return scatter plot. This helps investors understand the trade-off between potential returns and risk."

---

### Correlation Analysis (1:30 - 2:00)

**Visual:** Show the Correlation tab

**Narration:**
"The Correlation Analysis feature shows how stocks move relative to each other. This is crucial for portfolio diversification - stocks with low correlation provide better diversification benefits."

**Narration:**

## Recording Tips

1. **Screen Recording:** Use tools like OBS Studio, Loom, or built-in screen recording
2. **Resolution:** Record at 1080p for clear visibility
3. **Audio:** Use a good microphone for clear narration
4. **Pacing:** Speak clearly and allow time for viewers to see each feature
5. **Upload:** Upload to YouTube, Loom, or Mediasite as required by the assignment

## Key Points to Emphasize

- Interactive stock selection and time period adjustment
- Comprehensive statistical analysis
- Risk metrics and Sharpe ratio calculations
- Correlation analysis for portfolio diversification
- Data export functionality
- Clean, user-friendly interface
